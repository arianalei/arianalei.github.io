Projects

VEX VRC Push Back Robot Code
2025-2026 [Code Coming Soon]
A complete revamp of the robot code for this year's VEX VRC Push Back competition. This new codebase features significant improvements including integration of LemLib for much improved code modularity and sophisticated refined (curved) control. We also added odometry with both horizontal and vertical tracking wheels for precise position tracking and navigation. The new architecture provides better maintainability and more advanced autonomous capabilities compared to previous seasons!
VEX VRC High Stakes Robot Code
2024-2025 [Code]
As the lead coder for my VEX VRC team, I developed the driver control system and various autonomous routines. The robot features a 6-wheel tank drive system, optical color detection for ring sorting, and precise arm positioning with PID control. I implemented custom PID-based movement functions for distance-based movement, turning, and timing. The autonomous codes cover the first 15 seconds of alliance matches and dedicated 1-minute routines for skills challenges. The code supports 6 different autonomous routines selectable via the Brain screen, each optimized for different match strategies.
High Stakes Simulator
2024-2025 [Code]
I built a 2D robot simulator to help my VEX VRC team test our autonomous routines before running them on the actual robot. Using Python, pygame, and pygame_gui, we created an interactive visualization where we can see the robot move on the competition field, test different movement commands, and verify our sequences work correctly. It includes features like multiple starting positions, a command queue system, and a real-time timer to track our autonomous routine duration.
