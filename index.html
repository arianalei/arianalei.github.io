High Stakes Simulator
2024-2025 [Code]
I built a 2D robot simulator to help my VEX VRC team test our autonomous routines before running them on the actual robot. Using Python, pygame, and pygame_gui, we created an interactive visualization where we can see the robot move on the competition field, test different movement commands, and verify our sequences work correctly. It includes features like multiple starting positions, a command queue system, and a real-time timer to track our autonomous routine duration.
